# Control Estudiantina

Sistema de gestión para estudiantinas con las siguientes funcionalidades:

- 🎵 Gestión de cancionero
- 📅 Gestión de eventos
- 👥 Control de asistencia
- 💰 Gestión financiera
- 🚌 Gestión de transporte
- 🎟️ Venta de boletos

## Despliegue automático configurado
Este proyecto se despliega automáticamente en Firebase Hosting a través de GitHub Actions.
